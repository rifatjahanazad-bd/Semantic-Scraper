# AI-Powered HCP Data Extractor (LLM Web Scraper)

An experimental project that combines **Python**, **Playwright**, and a **local LLM (Qwen 2.5 via Ollama)** to extract structured healthcare professional (HCP) information from websites.

## Why I Made This

While collecting doctor and healthcare professional data, I noticed that traditional web scraping becomes difficult when every website has a different layout and structure. Writing custom scraping logic for each site is time-consuming and hard to maintain.

I built this project to explore whether a local LLM could act as an intelligent parser—taking raw webpage content and extracting useful information such as doctor names, roles, and departments without requiring site-specific rules.

## Project Status

⚠️ **Experimental / Work in Progress**

This project is primarily a learning and exploration project. I am currently learning **Python and Object-Oriented Programming (OOP)**, so the codebase is not yet structured the way I would like for a production-ready application.

The goal is to continuously improve the architecture, reliability, and extraction quality as my Python skills grow.
